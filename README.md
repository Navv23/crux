# CRUX: A Zero-Trust, Digital Polygraph System

**AI-Enabled Pre-Offer Candidate Verification Platform**

CRUX is an AI-powered pre-offer verification system designed to help organizations identify potential hiring fraud before an offer is released. It follows a Zero-Trust approach by treating all submitted candidate information as unverified until validated through multiple verification checks.

## Problem

Traditional background verification is slow, manual, and often takes several weeks. During this time, organizations risk losing quality candidates while remaining vulnerable to document fraud, identity manipulation, resume inflation, and employment inconsistencies.

## Solution

CRUX automates the candidate verification process by analyzing submitted information and generating a **Candidate Integrity Score (CIS)** along with an explainable verification report.

The platform assists recruiters in making faster, data-driven hiring decisions while reducing manual effort.

## Key Features

- AI-assisted candidate verification
- Candidate Integrity Score (CIS)
- Identity verification
- Employment history validation
- Education verification
- Document authenticity checks
- Resume and Job Description comparison
- Explainable verification reports
- Automated recruiter recommendations

## Workflow

1. Candidate uploads required documents.
2. CRUX validates submitted information.
3. Multiple verification checks are performed.
4. Candidate Integrity Score is calculated.
5. Recruiters receive a detailed verification report.

## Benefits

- Faster candidate screening
- Reduced hiring fraud
- Improved recruiter productivity
- Explainable AI-based decision support
- Consistent verification process
